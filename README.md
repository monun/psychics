현재 추가된 능력

flash(점멸) - 다이아몬드를 사용시 현재 마나의 10%만큼 순간이동합니다 이동 위치는 파티클로 표시됩니다.

mirror(응수) - 철 주괴를 우클릭할 시 1초간 응수 상태가 되어 공격력,공속,이속이 감소합니다.
이때 넉백에 면역이 생기는데, 넉백을 막을 시 디버프가 없어지고 1초간 공속,이속,공격력이 증가합니다.
